# Finac Bank - Personal Finance Tracker App

## Setup

1. Create Next Js project
2. Update layout.js & page.js
3. Create folder structure
4. Add shadcn to project

## Tech Used

1. NextJs
2. TailwindCss
3. Shadcn

## 1. Run this - to create NextJs project

```pnpm
pnpx create-next-app@lates
```
